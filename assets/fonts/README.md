# Bundled UI font

`InterVariable.ttf` is Inter 4.1 from the official Inter release:

- source: <https://github.com/rsms/inter/releases/tag/v4.1>
- upstream asset: `Inter-4.1.zip`
- upstream archive SHA-256: `9883FDD4A49D4FB66BD8177BA6625EF9A64AA45899767DDE3D36AA425756B11E`
- license: SIL Open Font License 1.1, reproduced in `LICENSE.txt`

The application embeds the variable font and selects Regular, Medium, and Semibold through its `wght` variation axis.
