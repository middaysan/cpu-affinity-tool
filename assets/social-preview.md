# Social Preview Brief

Create a repository social preview image with:

- size: `1280x640`
- product name: `CPU Affinity Tool`
- one clear UI screenshot from the app
- a visible `Windows` label
- 2 to 3 short value bullets

Suggested bullets:

- Save CPU affinity and priority rules
- Separate games from background apps
- Launch and re-apply settings from one tool

Keep the image technical and direct. Avoid benchmark claims, FPS guarantees, or cross-platform wording.
